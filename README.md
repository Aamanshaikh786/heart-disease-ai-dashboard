# 🫀 AI-Based Clinical Decision Support System for Cardiovascular Risk Prediction

## 📌 Overview
This project predicts early heart disease risk using machine learning and provides an interactive research-style dashboard built with Streamlit.

## 📊 Dataset
UCI Heart Disease Dataset (1025 samples)

## 🤖 Models Used
- Logistic Regression  
- Random Forest (Final Model)  
- XGBoost  

## 📈 Performance
- **Accuracy:** 98.5%  
- **AUC Score:** 1.0  

## 🖥 Features
- Risk percentage prediction  
- Clinical risk categorization  
- Model confidence  
- Feature importance visualization  
- Fully offline deployment  

## 🛠 Tech Stack
Python | Scikit-Learn | XGBoost | Streamlit | Pandas | Matplotlib
